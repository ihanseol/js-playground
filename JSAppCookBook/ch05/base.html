Image Machine dynamically generates JavaScript code and HTML needed to create image rollovers. Here's all you need to get started:

  • the number of image groups
  • the width and height in pixels
  • the image URLs
  • the URLs for the HREF attribute
  • optional text messages for the status bar
Image Machine then generates cut-and-paste code you can easily drop into your web pages. You can also see what the page looks like before you decide to keep the code.
